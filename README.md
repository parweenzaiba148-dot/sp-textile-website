# sp-textile-website
Business website for a textile shop with product collection and WhatsApp integration
This project is a business website developed for S.P Textile, a men's fabric shop based in Delhi.

The website showcases a wide range of premium fabric collections including Raymond, Siyaram's, and Chinar. It includes product listings with images and pricing, a clean user interface, and a responsive layout for mobile users.

Key Features:
- Product collection with 15+ designs
- Clean and modern UI
- WhatsApp integration for direct customer contact
- Mobile-friendly design
- Simple navigation for better user experience

This project is designed to help small local businesses establish an online presence and attract more customers.
